# Nums240075

Aplicação desenvolvida em C# com Windows Forms (.NET Framework) para a disciplina de PSI no âmbito do curso TGPSI - 11º ano.

## O que faz

A aplicação trabalha com números inteiros positivos e verifica se são **números 33-Zero-3**.

Um número é 33-Zero-3 quando:
- A soma dos seus algarismos é igual a **33**
- **Não contém** o algarismo 3

Exemplo: `9996` → 9+9+9+6 = 33, sem nenhum 3 → ✅ é 33-Zero-3

## Funcionalidades

- Inserir números e classificá-los na lista
- Detetar números repetidos
- Mostrar o menor número 33-Zero-3 inserido
- Calcular o somatório dos números 33-Zero-3
- Mostrar os números guardados ao fechar a aplicação

## Estrutura do projeto

- `Num33Zero3.cs` — classe que representa um número e verifica se é 33-Zero-3
- `Numeros.cs` — classe que gere a coleção de números 33-Zero-3
- `Form1.cs` — formulário principal com a interface da aplicação
